Site Purpose:

The purpose of this project is to apply the concepts learned during the BYU-Pathway program in real-world web development. By creating an online store, the project demonstrates essential programming skills such as:

  • Dynamic content rendering with JavaScript
  • DOM manipulation
  • Interactive UI design with CSS
  • Loading and using JSON data
  • Simulating real e-commerce functionality

This project also serves as a foundation for future development in full-stack web applications and entrepreneurial ventures in e-commerce.

Scenarios:

  • How can I place an order and pay securely?
  • How can I know if the product I want is still in stock and available for delivery in Brazzaville?
  • How can I contact customer service for questions or concerns?
  • Can I see more information or images about a product before buying it?
